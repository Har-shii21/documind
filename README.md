🤖 DocuMind — AI Documentation Generator

📌 Overview

DocuMind is an AI-powered documentation generator that helps developers automatically create project documentation from source code files.

The platform simplifies the process of writing README files, setup instructions, and project explanations using AI automation.

---

🚀 Features

- 📁 Upload source code files
- 🤖 AI-powered documentation generation
- 📄 Automatic README generation
- ⚡ Fast and simple user interface
- 📋 Copy generated documentation
- 💾 Download documentation as Markdown file

---

🛠️ Tech Stack

Frontend

- React.js
- CSS

Backend

- Flask (Python)

AI Integration

- Google Gemini API

---

⚙️ How It Works

1. User uploads a code file
2. The frontend sends code data to the backend
3. Flask backend processes the request
4. Gemini AI analyzes the code
5. Documentation is generated automatically

---

📂 Supported File Types

- HTML
- CSS
- JavaScript
- Python
- Java
- C++
- ZIP files

---

🎯 Problem Statement

Developers often spend significant time manually creating project documentation and README files.

DocuMind automates this workflow using AI, making documentation generation faster and more efficient.
